# shivrm.github.io

My website; uses Jekyll.

# How to use

1. Clone the repo
	```
	git clone https://github.com/shivrm/shivrm.github.io
	```

2. Navigate to the directory
	```
	cd shivrm.github.io
	```

3. Serve with Jekyll
	```
	jekyll serve
	```

4. Visit [http://localhost:4000](http://localhost:4000) in a browser of your choice.
